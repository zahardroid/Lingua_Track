# Changelog - Улучшения проекта LinguaTrack

## Версия 1.1 - Улучшения по ТЗ

### Добавлено

1. **Модель UserProfile** для связи пользователей с Telegram ID
   - Хранение telegram_id и telegram_username
   - Автоматическое создание профиля при первом использовании бота

2. **Ручное изменение даты следующего повторения**
   - Форма для изменения даты в карточке
   - URL: `/cards/<pk>/schedule/`
   - Доступно из детального просмотра карточки

3. **Режимы тестирования**
   - **Множественный выбор**: выбор правильного перевода из 4 вариантов
   - **Сопоставление**: drag-and-drop сопоставление слов и переводов
   - Новые URL: `/cards/test/multiple/`, `/cards/test/matching/`

4. **Экспорт/Импорт карточек**
   - Экспорт в CSV формат
   - Импорт из CSV с валидацией
   - URL: `/cards/export/`, `/cards/import/`

5. **Улучшенные напоминания в Telegram**
   - Работа с UserProfile для отправки напоминаний
   - Автоматическая отправка через Celery Beat

6. **Улучшенный requirements.txt**
   - Добавлены все необходимые зависимости
   - Указаны версии пакетов

### Улучшено

- Навигация в шаблоне base.html - добавлены dropdown меню для тестов и импорта/экспорта
- Обработка уровней при импорте - поддержка русских и английских названий
- Интеграция бота с UserProfile для корректной работы напоминаний

### Исправлено

- Удален дублирующий импорт bot в tasks.py
- Улучшена обработка ошибок при импорте карточек

## Версия 1.0 - Базовая версия

- CRUD для карточек
- Алгоритм SM-2 для интервального повторения
- Telegram-бот с базовыми командами
- Озвучка через gTTS
- Статистика и прогресс
